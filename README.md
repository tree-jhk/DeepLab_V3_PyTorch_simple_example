# 이미지 URL 변경 방법
해당 코드는 이미지를 불러오는 방법으로 URL을 이용합니다.  

이미지 파일(PNG, JPG, 등)을 URL로 바꾸는 방법:  

[https://ifh.cc/] 접속 → 이미지 업로드 → 이미지 주소 복사(공유 주소 x)

URL로 변경하면, 변경된 URL에서는 이미지 제목이 바뀝니다. 바뀐 제목으로 사용합니다.
